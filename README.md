# Tetris
Tetris environment forked by [source](https://github.com/michiel-cox/Tetris-DQN.git)

## Action for human testing

| Keyboard  | Move |
| ---: | ---: |
| Right | shift right 1 block |
| Left | shift left 1 block |
| Space | drop down |
| Shift | hold |
| r | restart the game |


## How to run
Run `run_human.py` to play Tetris manually.
